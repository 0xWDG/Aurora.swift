Replacing

  • replacingMatches(in:with:count:)

    Returns a new string where each substring matched by regex is replaced with template.

    The template string may be a literal string, or include template variables: the variable $0 will be replaced with the entire matched substring, $1 with the first capture group, etc.

    For example, to include the literal string “$1” in the replacement string, you must escape the “$”: \$1.

    Declaration

    Swift

    @inlinable
public func replacingMatches(in input: String, with template: String, count: Int? = nil) -> String

    Parameters

    regex

    A regular expression to match against self.
    template

    A template string used to replace matches.
    count

    The maximum count of matches to replace, beginning with the first match.

    Return Value

    A string with all matches of regex replaced by template.
